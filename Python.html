Collezione di Dati

La collezione di Dati è un argomento molto importante di Python, dove vediamo come creare un "contenitore" di variabili

Indice

  • Torna all'indice
  • Liste
  • Tuple
  • Set
  • Dictionary

Una collezione di dati sono tutti quei valori che raggruppano un numero di dati elevato e abbiamo visto che esistono diversi tipi: LIST, TUPLE, SET, DICTIONARY

  • Le liste sono collezioni ordinate e modificabili. Permettono duplicati.
  • Le tuple sono collezioni ordinate ma immutabili. Permettono duplicati.
  • I set sono collezioni non ordinate e perciò non indicizzate. Non permettono duplicati.
  • I dictionary sono collezioni ordinate e modificabili. Non permetono duplicati.

In che senso? Per Python capire che la persona di cui si sta parlando, 
sia la persona1 e non una persona generica a caso, 
bisogna che si specifichi che "self" corrisponda a quell'istanza dell'oggetto 
che deriva da Persona. Se così non fosse, sarebbe la stessa come chiedere a persona1(Luca Rossi): 
"Ciao come ti chiami?" "Giggio" quando tu non sei Giggio, ma Luca.


class Persona:
    def __init__(self, nome, cognome): #Self è il riferimento a sè stesso. l'ISTANZA perciò, si 
                                        #riferisce a sè stesso (persona1 o persona2).
    self.nome = nome
    self.cognome = cognome

persona1 = Persona("Luca", "Rossi")
persona2 = Persona("Marco", "Verdi")

print(persona1.nome) #Luca
print(persona2.nome) #Marco

# L'init della classe Insegnante andrà a fare l'override del init di persona. 
# L'init che vale perciò è il secondo di insegnante. 
# Stessa cosa succede anche con i metodi chiamati con lo stesso nome ad esempio
# Se volessi "portare" ad insegnante, gli stessi tratti di persona e quindi
# Nome, Cognome e magari aggiungere anche qualcosa in più farò così:

class Persona:
    def __init__(self, nome, cognome): 
    self.nome = nome
    self.cognome = cognome
    
    def saluta(self):
        print("Ciao sono" + self.nome)

class Insegnante(Persona): 
    def __init__(self, nome, cognome, materia):
        super().__init__(nome, cognome) # Super sta ad indicare la Classe Madre (quella super)
        self.materia = materia
    
    def materia(self):
        print("Io insegno la" + self.materia)

    def dai_voti(self):
        print("Hai preso 8")

persona1 = Persona("Luca", "Rossi") 
insegnante1 = Insegnante("Mario", "Bianchi", "Matematica") 

insegnante1.saluta() # Ciao sono Mario
insegnante1.materia() # Io insegno la matematica
insegnante1.dai_voti() # Hai preso 8

Scope

  • Torna all'indice

In questo paragrafo vedremo lo Scope in Python, come vengono definiti e come funzionano:

Lo scope è la disponibilità delle variabili nel mio ambiente. 
Noi abbiamo delle variabili fuori le funzioni ma anche al suo interno:

x = 300

def funzione():
    y = 400


print(x) # 300
print(y) # ❌ # Perchè si trova all'interno di una funzione

def funzione():
    y = 400
    print(y)

funzione() # ✅ 400

Quindi abbiamo scope globali e scope locali. Le variabili all'interno di una funzione
sono variabili locali, mentre quelle al di fuori, sono globali

x = 400

def funzione():
    print(x)

funzione() # ✅ 400

-------

def funzione():
    x = 400
    def metti_numero():
        print(x)
    metti_numero()

funzione() # ✅ 400

-------------------- KEYWORD GLOBAL --------------------

x = 400

def funzione():
    global x # Fa riferimento alla X globale, quindi alla X = 400
    x = 100 # OVERRIDE
    print(x)

funzione() # 100
print(x)   # 100 e non 400 perchè c'è stato  un override della globale.

Lavorare con i File

  • Torna all'indice

In questo paragrafo vedremo come lavorare con i File in Python:

  • r - Read: Apre il file per leggere, errore se non esiste
  • a - Append: Apre il file per appendere, crea il file se non esiste
  • w - Write: Apre il file per scrivere, crea il file se non esiste
  • x - Create: Crea il file, errore se non esiste

Creeremo ad esempio un File di testo.txt per il File Handling.

f = open("testo.txt", "r") / f = open("testo.txt") # E' la stessa cosa, di default c'è R
print(f.read()) # Legge il file.
print(f.read(5)) # Solo 5 parole
print(f.readline()) # Legge solo la prima riga
-----------------------------------------------------
f = open("testo.txt", "a")
f.write("Dentro ci sto scrivendo quello che voglio")
f.close()
-----------------------------------------------------
f = open("testo.txt", "w")
f.write("Dentro ci sto scrivendo quello che voglio") # Andrà a sovrascrivere tutto il testo
f.close()
-----------------------------------------------------

f = open("createFile.txt", "x") # Creazione del file.
